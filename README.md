# Git Version Control Setup

## 📖 Overview

This repository demonstrates the complete setup of Git for version control and GitHub for remote repository management. The project includes a simple HTML webpage used to practice essential Git operations such as initialization, staging, committing, and pushing code to a remote repository.

## 🎯 Objectives

* Install and configure Git
* Create and initialize a local repository
* Track project files using Git
* Commit source code changes
* Connect the repository to GitHub
* Push code to a remote repository

## 🛠️ Technologies Used

* GitHub
* HTML5

## 🚀 Steps Performed

### 1. Git Installation

Installed Git and verified the installation using:

```bash
git --version
```

### 2. Git Configuration

Configured Git user details:

```bash
git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
```

### 3. Repository Initialization

```bash
git init
```

### 4. Staging Files

```bash
git add .
```

### 5. Creating the First Commit

```bash
git commit -m "Initial commit"
```

### 6. Connecting to GitHub

```bash
git remote add origin <repository-url>
```

### 7. Pushing to GitHub

```bash
git branch -M main
git push -u origin main
```
